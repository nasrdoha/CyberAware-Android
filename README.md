# CyberAware – Android Cybersecurity Awareness Application

CyberAware is an Android application developed as a graduation project to improve cybersecurity awareness through interactive quizzes and real-world attack scenarios.

## Features

- Interactive cybersecurity quizzes
- Automatic difficulty assignment
- Guest mode
- 25-second timer for each question
- Real-time feedback
- Final score and personalized recommendations
- Arabic and English language support

## Technologies

- Java
- Android Studio
- XML

## Purpose

The application helps users improve their awareness of cybersecurity topics such as phishing, password security, malware, and social engineering through an interactive learning experience.
